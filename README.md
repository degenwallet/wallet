## 🪄 Features

- 👨‍👩‍👧‍👦 **Open Source & Community Owned** with web3 ethos.
- 🔑 **Secure** and **Privacy** preserving wallet.
- 📱 **Cross Platform** build with React Native (TypeScript) for iOS and Android.

2. Run `npm install && npm install:ios`
